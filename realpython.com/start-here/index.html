Here’s an example:

Let’s say you want to learn about Python Decorators.

They’re an amazingly useful feature in Python, but can appear quite arcane at first:

Python 
def do_twice(func):
    def wrapper_do_twice(*args, **kwargs):
        func(*args, **kwargs)
        func(*args, **kwargs)
    return wrapper_do_twice

@do_twice
def greet(name):
    print(f"Hello, {name}!")

>>> greet("World")
